## Publishing Options

### Easiest: GitHub Pages
1. Create GitHub account
2. Create new repository (public)
3. Upload all files
4. Settings → Pages → Select main branch
5. Done! URL: `https://username.github.io/repo-name/`

### Also Easy: Netlify Drop
1. Go to netlify.com/drop
2. Drag your project folder
3. Get instant URL

## After Publishing

1. Test the live URL on your phone
2. Share with study group peers
3. Share with 1-2 target audience members
4. Gather feedback on:
   - Can they find date/time/location easily?
   - Is it clear what each session covers?
   - Does it look professional?
5. Make revisions based on feedback
6. Submit final URL to instructor
